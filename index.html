Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, data was pulled from the OpenWeatherMap API to assemble a dataset on over 500 cities across the world. After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs. latitude. The factors included temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of analysis, as well as explantions and descriptions of any trends and correlations witnessed.
